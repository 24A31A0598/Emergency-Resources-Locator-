# 🚨 Emergency Resource Locator

**Emergency Resource Locator** is a smart and user-friendly web application designed to help users quickly find nearby essential public services such as hospitals, police stations, fire stations, pharmacies, and more.

---

## 📌 Overview

Emergency Resource Locator simplifies access to critical services by allowing users to detect their current location using GPS or search for a place manually. It provides a list of nearby emergency resources along with an interactive map for easy navigation.

The platform focuses on improving response time during emergencies while ensuring a smooth and accessible user experience for everyone.

---

## ✨ Key Features

✔ GPS-based current location detection  
✔ Manual location search functionality  
✔ Nearby emergency services listing  
✔ Interactive map integration using Leaflet  
✔ Quick access to emergency contact options  
✔ Categorized services (Hospital, Police, Fire, Pharmacy, etc.)  
✔ Clean and responsive user interface  
✔ Fast and lightweight web application
